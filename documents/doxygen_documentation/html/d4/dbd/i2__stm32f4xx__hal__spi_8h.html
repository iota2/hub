SPI send / receive.

Transmits and receives data on SPI bus, This function will asserts and deasserts the CS pin.

Parameters
[in]*instSPI instance.
[in]data_widthSize of data bits for SPI SPI data width..
[in]clk_speedSPI clock speed SPI clock speed..
[in]spi_modeSPI communication mode SPI Operation Modes..
[in]first_bitCommunication start bit SPI First bit to send / receive..
[in]*txbufIf null, then this function will only receive data on SPI bus, and put it in rxbuf, else this buffer will be transmitted on SPI bus.
[out]*rxbufIf null, then this function will only transmit data form txbuf on SPI bus, else this buffer will be used to receive data on SPI bus.
[in]sizeAmount of data to be transmitted / received.
[in]timeoutSPI bus HAL timeout.
Returns
Execution error code iota2 unified Error Codes..
Note
If both txbuf and rxbuf are non null, then txbuf will be transmitted TX pin and on same clock data will be received on RX pin and will be put in rxbuf.
